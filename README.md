# العوجان للسياحة والسفر | Al-Awajan Travel

تطبيق متطور لحجز الرحلات الدولية والطرود، مصمم بهوية بصرية فاخرة وأحدث تقنيات الذكاء الاصطناعي.

## المخطط التصميمي (App Blueprint)

### 🎨 الهوية البصرية
- **الألوان:** أخضر زمردي ملكي (Primary: #003d2d) وذهبي كلاسيكي (Accent: #b08d40).
- **الخطوط:** Noto Sans Arabic (للغة العربية) و Inter (للأرقام).
- **التخطيط:** يدعم RTL بالكامل، تصميم Mobile-first، حواف مستديرة (2xl)، وظلال ناعمة.

### 🚀 المميزات الرئيسية
1. **البحث والحجز:** نظام متكامل لاختيار الرحلات والمقاعد التفاعلية.
2. **تتبع الطرود:** تتبع مسار الشحنات بين المدن مع شريط تقدم بصري.
3. **التتبع المباشر:** مراقبة موقع الحافلة والوقت المتوقع للوصول.
4. **المساعد الذكي:** مساعد AI مدعوم بـ Genkit للإجابة على الاستفسارات باللغة العربية.
5. **الملف الشخصي:** إدارة الحجوزات ونظام العضوية الذهبية.

### 🛠 التقنيات
- **Frontend:** Next.js 15 (App Router), Tailwind CSS, Shadcn UI.
- **Backend:** Firebase (Firestore, Auth).
- **AI:** Google Gemini via Genkit.
- **Icons:** Lucide React.
